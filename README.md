# Music Trading App

WebApp (frontend + backend with api) for trading music online.

### Run

Mongodb is required and should be configured in environment configuration

```
server/config/environment/
```

Install dependencies

```
npm i && bower i
```

Run server locally

```
grunt serve
```

### Uses

* Grunt
* Node
* Express
* Apple Itunes API
* MongoDB
* Angular
* SCSS
